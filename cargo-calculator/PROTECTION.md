# Защита 3D-калькулятора Правильные Грузчики

## Установленные меры защиты

### 1. Domain Lock + Referer + Iframe Protection
- Калькулятор работает **только** на разрешённых доменах:
  - `pragruz.ru`
  - `www.pragruz.ru`
  - `pravilnye-gruzchiki.ru`
  - `localhost` / `127.0.0.1`

- **Дополнительные проверки**:
  - **Referer** — если пользователь пришёл с другого сайта, доступ блокируется
  - **Iframe** — калькулятор нельзя встраивать в `<iframe>` на сторонних сайтах

- При любом нарушении — показывается красивая страница блокировки с кодом нарушения.

### 2. Обфускация кода (vite-plugin-obfuscator)
- Включена агрессивная обфускация при сборке
- Самозащита кода (`selfDefending`)
- Запрет вывода в консоль
- Защита от DevTools

### 3. Watermark
- В код встроен уникальный идентификатор сборки:
  ```
  PG-CARGO-3D-v2026-07-19-019f78c0
  ```

### 4. Блокировка DevTools
- Отслеживание открытия DevTools
- Блокировка клавиш: `F12`, `Ctrl+Shift+I/J/C`, `Ctrl+U`
- Блокировка правой кнопки мыши на canvas

### 5. Дополнительно
- Отключены Source Maps
- Console.log удаляются в production
- Защита от копирования canvas

---

## Как собрать защищённую версию

```bash
cd cargo-calculator
npm install
npm run build
```

Собранные файлы будут находиться в `dist/`.

---

## Рекомендации

1. **Никогда** не публикуй `src/` на публичный сервер
2. Всегда используй `npm run build` перед деплоем
3. Регулярно меняй `WATERMARK` в `protect.ts`
4. При смене домена — обнови `ALLOWED_DOMAINS`
