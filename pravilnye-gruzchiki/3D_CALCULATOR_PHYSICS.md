# Физика и Взаимодействие Предметов - 3D Калькулятор

## Дата: 2026-07-18

## Обзор Системы

3D калькулятор использует продвинутый алгоритм размещения предметов с учётом:
- **Приоритета** (тяжёлые и крупные предметы размещаются первыми)
- **Штабелирования** (какие предметы можно ставить друг на друга)
- **Веса** (ограничение на весstacked предметов)
- **Габаритов** (автоматический поворот для оптимального размещения)
- **Хрупкости** (запрет на укладку на бок для некоторых предметов)

## Алгоритм Размещения

### 1. Приоритет Сортировки

Предметы размещаются в порядке приоритета:

| Приоритет | Тип | Примеры |
|-----------|-----|---------|
| **1** | Самые тяжёлые | Холодильник (80kg) |
| **2** | Очень тяжёлые | Сейф (180kg), Пианино (210kg) |
| **3** | Крупная мебель | Диван, Кровать, Шкаф, Стол, Комод |
| **4** | Средние предметы | Стиралка, Стулья, ТВ, Велосипед, Микроволновка, Принтер |
| **5** | Паллеты и оборудование | Паллета, Стеллаж, Торговое оборудование |
| **6** | Мелкие предметы | Коробки, Тумбочки, Проектор |

### 2. Автоматический Поворот

Длинные предметы автоматически поворачиваются на 90° для размещения вдоль стены:
- Диваны (прямой и зоны ожидания)
- Кровати (обычная и детская)
- Велосипеды
- Коляски
- Столы (обеденный и рабочий)
- Комоды
- Стеллажи
- Шкафы для документов

**Логика поворота:**
```javascript
// Предмет поворачивается если он в списке длинных предметов
const isLongItem = "sofa"===kind || "bed"===kind || "table"===kind || ...;
if (isLongItem) {
  rotation = [rotation[0], Math.PI/2, rotation[2]]; // Поворот на 90°
}
```

### 3. Укладка на Бок

Предметы с `canLaySide: true` могут быть уложены на бок если:
- Их высота превышает высоту кузова
- Это позволяет сэкономить место

**Могут быть уложены на бок (21 предмет):**
- Мебель: диваны, шкафы, кровати, столы, стулья, комоды, тумбочки
- Техника: микроволновка
- Прочее: велосипеды, коляски, коробки, стеллажи

**НЕ могут быть уложены на бок (12 предметов):**
- Холодильник (может сломаться компрессор)
- Стиральная машина (барабан)
- Телевизор (экран)
- Пианино (механизм)
- Сейф (конструкция)
- Растения (земля высыпется)
- Мониторы, Принтеры, Серверы, Кофемашины, Проекторы (электроника)
- Паллеты (конструкция)

### 5. Проверка Коллизий

Алгоритм проверяет пересечение с уже размещёнными предметами:

```javascript
// Для каждого нового предмета проверяем все размещённые
for (const placedItem of placedItems) {
  // Проверяем пересечение по X, Y, Z
  if (newItem.maxX > placedItem.minX && 
      newItem.minX < placedItem.maxX &&
      newItem.maxZ > placedItem.minZ && 
      newItem.minZ < placedItem.maxZ &&
      newItem.maxY > placedItem.minY && 
      newItem.minY < placedItem.maxY) {
    // Коллизия! Ищем другую позицию
  }
}
```

## Как Пользователь Взаимодействует с Системой

### Добавление Предметов

1. **Из библиотеки переезда:**
   - Выберите тип переезда (квартирный/офисный/коммерческий)
   - Нажмите на предмет из отфильтрованной библиотеки
   - Предмет автоматически размещается в кузове

2. **Через пресеты:**
   - Выберите готовый пресет (1-комнатная, 2-комнатная, офис и т.д.)
   - Система автоматически добавит типовой набор предметов

3. **Автозаполнение коробками:**
   - Нажмите "📦 Заполнить коробками"
   - Система заполнит свободное место коробками для наглядности

### Управление Инвентарём

**Панель "Инвентарь" (слева):**
- Показывает все добавленные предметы
- Можно удалить предмет (кнопка ✕)
- Можно выбрать предмет для редактирования позиции

**Панель "Загрузка и цена" (справа):**
- Показывает общую статистику:
  - Количество предметов
  - Общий объём (м³)
  - Общий вес (kg)
  - Процент заполнения кузова
  - Рекомендуемый транспорт
  - Стоимость переезда

### Автоматическое Размещение

Система автоматически:
1. Сортирует предметы по приоритету
2. Размещает тяжёлые предметы на полу
3. Поворачивает длинные предметы вдоль стены
4. Штабелирует предметы с учётом ограничений по весу
5. Проверяет коллизии
6. Если предмет не помещается → добавляет в overflow (переполнение)

### Переполнение (Overflow)

Если предметы не помещаются в один кузов:
- Система показывает количество предметов в overflow
- Рассчитывает необходимый объём и вес для дополнительных рейсов
- Предлагает увеличить количество машин или выбрать更大ий транспорт

## Технические Детали

### Функции Размещения

**`ae(items, vehicleType)`** - основная функция упаковки
- Принимает массив предметов и тип транспорта
- Возвращает `{placed: [], overflow: []}`

**`de(items, vehicleType, idGenerator, iterations)`** - автозаполнение
- Заполняет свободное место коробками
- Итеративно добавляет коробки пока есть место

**`me(items, newItem, vehicleType)`** - добавление одного предмета
- Проверяет коллизии
- Находит оптимальную позицию
- Возвращает обновлённый массив предметов

### Проверка Штабелирования

```javascript
// Проверяем можно ли поставить предмет на другой
if (placedItem.stackable && newItem.weight <= placedItem.maxStackWeight) {
  // Можно поставить сверху
  newItem.position[1] = placedItem.maxY; // Y координата = верх предмета
} else {
  // Нельзя, ищем другую позицию
}
```

### Расчёт Объёма и Веса

```javascript
// Объём предмета
volume = dimensions.length × dimensions.width × dimensions.height

// Вес предмета (с коробками для паллет)
weight = item.weight + sum(boxes.map(box => box.weight))

// Общий объём и вес
totalVolume = sum(items.map(item => volume(item)))
totalWeight = sum(items.map(item => weight(item)))
```
