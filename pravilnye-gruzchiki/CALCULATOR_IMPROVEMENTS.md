## 🎯 ПРИОРИТЕТ 1: Конверсия в Заявки

Эти улучшения напрямую влияют на получение лидов.

### 1.1 Кнопка "Оставить заявку" с предзаполненными данными ⭐⭐⭐

**Проблема:** Пользователь рассчитал стоимость, но не знает как связаться.

**Решение:** Кнопка которая открывает модальное окно с:
- Предзаполненным текстом: "Хочу заказать переезд: 2-комнатная квартира, Краснодар → Сочи, ~12 м³, ~900 кг, Газель 12 м³, ~4500₽"
- Кнопками: WhatsApp, Telegram, Телефон
- Полем для имени и телефона

**Реализация:** ~50 строк кода
```javascript
// Данные уже есть в store
const payload = {
  from, to, distance,
  moveType, vehicleType,
  totalVolume, totalWeight,
  totalPrice
};

// Генерируем текст
const message = `Заявка на переезд:\n${from} → ${to} (${distance} км)\n` +
  `Объём: ${totalVolume} м³, Вес: ${totalWeight} кг\n` +
  `Транспорт: ${vehicle}\n` +
  `Стоимость: ~${totalPrice}₽`;

// Открываем WhatsApp
window.open(`https://wa.me/79001234567?text=${encodeURIComponent(message)}`);
```

**Вес:** +2 KB  
**Сложность:** 1 час  
**ROI:** ⭐⭐⭐ Высокий (прямые заявки)

### 1.2 Скриншот 3D сцены для заявки ⭐⭐⭐

**Проблема:** Пользователь хочет показать что именно он перевозит.

**Решение:** Кнопка "Сделать скриншот" которая:
- Делает скриншот canvas через `canvas.toDataURL()`
- Сохраняет как PNG
- Можно приложить к заявке

**Реализация:** ~20 строк кода
```javascript
function takeScreenshot() {
  const canvas = document.querySelector('canvas');
  const link = document.createElement('a');
  link.download = `pereezd-${Date.now()}.png`;
  link.href = canvas.toDataURL('image/png');
  link.click();
}
```

**Вес:** +0.5 KB  
**Сложность:** 30 минут  
**ROI:** ⭐⭐⭐ Высокий (визуальная заявка)

## 💡 ПРИОРИТЕТ 2: Улучшение UX

Эти улучшения делают калькулятор удобнее.

### 2.1 Этажность и Лифт ⭐⭐⭐

**Проблема:** Цена не учитывает сложность погрузки (5 этаж без лифта vs 1 этаж с лифтом).

**Решение:** Добавить в расчёт:
- Этаж отправления (1-9+)
- Этаж прибытия (1-9+)
- Наличие лифта (да/нет)

**Формула:**
```javascript
const floorSurcharge = (fromFloor + toFloor) * (hasElevator ? 50 : 150);
// 1 этаж с лифтом: +100₽
// 5 этаж без лифта: +1500₽
```

**Реализация:** ~40 строк кода  
**Вес:** +2 KB  
**Сложность:** 2 часа  
**ROI:** ⭐⭐⭐ Высокий (точная цена)

---

### 2.2 Горячие Клавиши ⭐⭐

**Проблема:** Опытные пользователи хотят быстрее работать.

**Решение:**
- `Delete` — удалить выбранный предмет
- `R` — повернуть на 90°
- `Ctrl+Z` — отменить
- `Ctrl+Shift+Z` — повторить
- `Space` — снять выделение
- `1-3` — выбор оси перемещения

**Реализация:** ~30 строк кода
```javascript
useEffect(() => {
  const handler = (e) => {
    if (e.key === 'Delete' && selectedPalletId) {
      removePallet(selectedPalletId);
    }
    if (e.key === 'r' || e.key === 'R') {
      rotateSelectedY();
    }
    if (e.ctrlKey && e.key === 'z') {
      undo();
    }
  };
  window.addEventListener('keydown', handler);
  return () => window.removeEventListener('keydown', handler);
}, [selectedPalletId]);
```

**Вес:** +1 KB  
**Сложность:** 1 час  
**ROI:** ⭐⭐ Средний (удобство для power users)

### 2.3 Туториал / Подсказки ⭐⭐

**Проблема:** Новые пользователи не знают как начать.

**Решение:** При первом запуске показать 3 подсказки:
1. "Выберите тип переезда или добавьте предметы вручную"
2. "Кликните на предмет чтобы переместить его"
3. "Нажмите 'Заполнить коробками' для автоматического заполнения"

**Реализация:** ~50 строк кода
```javascript
const [showTutorial, setShowTutorial] = useState(
  !localStorage.getItem('tutorial_seen')
);

if (showTutorial) {
  return <TutorialOverlay steps={tutorialSteps} onClose={() => {
    localStorage.setItem('tutorial_seen', '1');
    setShowTutorial(false);
  }} />;
}
```

**Вес:** +3 KB  
**Сложность:** 2 часа  
**ROI:** ⭐⭐ Средний (снижает bounce rate)

## 🔧 ПРИОРИТЕТ 3: Технические Улучшения

Эти улучшения не видны пользователю, но улучшают код.

### 3.1 Ленивая Загрузка 3D Сцены ⭐⭐⭐

**Проблема:** 3D сцена загружается сразу, даже если пользователь ещё не готов.

**Решение:** Загружать Three.js только после клика "Начать расчёт":
```javascript
const [sceneLoaded, setSceneLoaded] = useState(false);

if (!sceneLoaded) {
  return <StartScreen onStart={() => setSceneLoaded(true)} />;
}

return <Canvas>...</Canvas>;
```

**Вес:** -50 KB начальной загрузки  
**Сложность:** 3 часа  
**ROI:** ⭐⭐⭐ Высокий (быстрее первая загрузка)

---

### 3.2 Кэширование Расчётов ⭐⭐

**Решение:** Кэшировать результат на 500ms:
```javascript

const lastCalcRef = useRef(0);

function calculatePrice() {
  const now = Date.now();
  if (now - lastCalcRef.current < 500) {
    return cachedPrice;
  }
  lastCalcRef.current = now;

  setCachedPrice(result);
}
```

**Вес:** +0.5 KB  
**Сложность:** 1 час  
**ROI:** ⭐⭐ Средний (производительность)

## 🎨 Альтернативные Идеи (Не Рекомендуются)

Эти идеи могут утяжелить код или усложнить UX:

### ❌ Чат с оператором
- **Проблема:** Требует backend, WebSocket, модерацию
- **Вес:** +50 KB (виджет) + сервер
- **Альтернатива:** Кнопка WhatsApp (уже есть)

### ❌ 3D модели всех предметов
- **Проблема:** 33 модели × 100 KB = 3.3 MB
- **Вес:** +3300 KB
- **Альтернатива:** Текущие геометрические примитивы работают хорошо

### ❌ Мультиязычность
- **Проблема:** Нужен перевод всех 33 предметов + UI
- **Вес:** +20 KB (файлы переводов)
- **Альтернатива:** Целевая аудитория — русскоязычная

### ❌ AR режим (дополненная реальность)
- **Проблема:** Требует WebXR, сложные библиотеки
- **Вес:** +200 KB
- **Альтернатива:** 3D сцена уже даёт хорошее представление

### ❌ Социальные функции (поделиться расчётом)
- **Проблема:** Уже есть share link
- **Вес:** +5 KB (кнопки соцсетей)
- **Альтернатива:** Share link достаточно

## 📝 Технические Заметки

### Как избежать утяжеления:

1. **Использовать существующие библиотеки**
   - Three.js уже загружен → используем для скриншотов
   - Zustand уже есть → храним промокоды в store
   - LocalStorage уже используется → сохраняем туториал

2. **Ленивая загрузка**
   - Загружать 3D сцену только после клика
   - Загружать PDF экспорт только при клике на кнопку
   - Загружать изображения по требованию

3. **Минификация**
   - Текущий код уже минифицирован (147 KB)
   - Использовать tree-shaking для удаления неиспользуемого кода
   - Сжимать изображения (WebP вместо PNG)

4. **Кэширование**
   - Service Worker для кэширования статических ресурсов
   - LocalStorage для сохранения состояния
   - Memory cache для частых расчётов
