## ⚠️ Что НЕ сделано (задачи для следующего агента)

### Приоритет ВЫСОКИЙ

1. **Блог: ещё 5-10 статей** — план предполагал 20-30 статей, сейчас 20. Нужны:
   - Кейс: такелажные работы (станки, сейфы, пианино)
   - Кейс: разнорабочие на склад/стройку
   - Сравнение: самостоятельный переезд vs грузчики
   - Инструкция: как перевезти холодильник/стиралку
   - Аренда газели: когда Газель не справится (нужна ЗИЛ/Камаз)
   - Сезонность цен в Краснодарском крае

2. **Снижение Jaccard overlap (~76%)** — главный источник: общие шаблонные блоки (advantages, calculator, form) идентичны на всех страницах. Решения:
   - Добавить уникальные параграфы в преимущества для каждой услуги×города
   - Калькулятор: добавить поясняющий текст, специфичный для услуги
   - Форма: менять заголовок/подзаголовок под контекст страницы

3. **Lighthouse/PageSpeed аудит** — не проводился (нет Chrome в среде). Пользователю нужно запустить локально:
   - Performance, Accessibility, Best Practices, SEO
   - Особое внимание: CLS (изображения с width/height должны помочь), LCP (hero images)

### Приоритет СРЕДНИЙ

4. **Реальные фотографии** — на сайте только стоковые изображения. Нужны:
   - Реальные фото бригад за работой
   - Фотографии грузовиков/Газелей с логотипом
   - Фото до/после переездов
   - Требует участия владельца сайта

5. **Custom OG images для блога** — сейчас все статьи используют generic og:image. Нужно:
   - Создать уникальные og:image (1200×630) для каждой статьи
   - Или хотя бы использовать city-specific hero-изображения

6. **Расширение about.html** — можно добавить:
   - Лицензии и сертификаты (если есть)
   - Детальные условия компенсации при повреждении
   - Отзывы с фотографиями (реальные)
   - Видео-презентация компании

7. **Template pages в partials/** — содержат `{{BASE}}` плейсхолдеры, не используются в проде, но лежат в репо. Можно:
   - Удалить если не планируются
   - Или настроить сборку через шаблонизатор

### Приоритет НИЗКИЙ

8. **Микроразметка Review/AggregateRating** — можно добавить звёздочки в сниппет Google
9. **BreadcrumbList schema** — для навигационных хлебных крошек в SERP
10. **VideoObject schema** — если появятся видео
