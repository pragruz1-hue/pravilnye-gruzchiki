# Эталон редизайна главной

**Файл-эталон:** `preview-homepage-redesign.html` (PREVIEW v8+)  
**Общие стили:** `css/11-homepage-redesign.css`  
**Отзывы:** `js/modules/reviews.js` — ~10 на город, **удвоение массива** для loop + **pause on hover**

## Структура блоков (порядок)
1. Header (без «Междугородние» в меню; звонки 8:00–20:00 · работы 24/7)
2. Hero (H1 грузчики/переезды, FOMO без подписи «ориентировочные…», форма «Быстрый расчёт»)
3. Stats (count-up с 0)
4. Наша команда (фото + 4 quality-карточки, без «✓ дешёвых»)
5. Преимущества premium 01–06
6. Калькулятор
7. Как проходит заказ (hover desktop / scroll mobile)
8. Услуги (оригинальные карточки)
9. Отзывы (карусель, 10×2, service tags)
10. Для бизнеса (усиленный B2B)
11. Автопарк (2 газели, фото сверху, note про крупный транспорт)
12. Партнёры
13. Форма + контакты
14. География (клик → модалка)
15. Локальный SEO bento (уникально по городу)
16. FAQ premium
17. Финальный CTA
18. Footer полный

## Убрано с эталона
- Быстрый выбор / сценарии
- Междугородний блок на главной
- «Бригады в форме» / «славянской…» / «фейковых филиалов»
- FOMO-подпись «Ориентировочные показатели…»
- «База» на карточке Краснодара
