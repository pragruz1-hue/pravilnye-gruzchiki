### ✅ 1.2 Скриншот 3D сцены

**Что добавлено:**
- Функция `takeScreenshot()` делает скриншот canvas
- Кнопка "📸 Скриншот" в панели цены
- Автоматическое сохранение как PNG с именем `pereezd-{timestamp}.png`

**Реализация:**
```javascript
function takeScreenshot() {
  const canvas = document.querySelector('canvas');
  canvas.toBlob(blob => {
    const url = URL.createObjectURL(blob);
    const link = document.createElement('a');
    link.download = `pereezd-${Date.now()}.png`;
    link.href = url;
    link.click();
    URL.revokeObjectURL(url);
  }, 'image/png', 0.95);
}
```

**Вес:** +0.5 KB  
**ROI:** ⭐⭐⭐ Высокий (визуальная заявка)

**Реализация:**
```javascript
useEffect(() => {
  const handler = e => {
    const state = Pe.getState();
    if (e.target.tagName === "INPUT" || e.target.tagName === "TEXTAREA") return;
    if (e.key === "Delete" && state.selectedPalletId) {
      state.removePallet(state.selectedPalletId);
      trackEvent("item_deleted");
    }
    // ... другие клавиши
  };
  window.addEventListener("keydown", handler);
  return () => window.removeEventListener("keydown", handler);
}, []);
```
