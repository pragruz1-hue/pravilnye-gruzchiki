# YML-фиды по регионам для Яндекса

Основной фид: https://pragruz.ru/feed.xml
Региональный фид: https://pragruz.ru/feeds/feed-krasnodarskiy-krai.xml

Мульти-региональные фиды дальних городов удалены: сайт сфокусирован на Краснодарском крае. Москва, Санкт-Петербург, Ростов-на-Дону и другие города используются только как направления междугородних переездов, а не как локальные филиалы.
