# YML-фиды по регионам для Яндекса

Основной и единственный фид для загрузки в Яндекс Вебмастер:
https://pragruz.ru/feed.xml

Файл https://pragruz.ru/feeds/feed-krasnodarskiy-krai.xml оставлен только как архив предыдущей версии. Не регистрируйте оба URL одновременно: предложения с одинаковыми URL не должны дублироваться в нескольких фидах одного региона.

Мульти-региональные фиды дальних городов удалены: сайт сфокусирован на Краснодарском крае. Москва, Санкт-Петербург, Ростов-на-Дону и другие города используются только как направления междугородних переездов, а не как локальные филиалы.
