# Журнал изменений feed.xml

## Версия 1.3 — 2026-07-13 — Финальная версия: цены, удаление аутсорсинга, фикс krasnodar/index.html

### Итоговые изменения:

1. **Такелаж (rigging):** цена `1200` → `1000` (совпадает с JSON-LD на страницах)
2. **Аутсорсинг (outsourcing):** полностью удалён из фида (5 офферов)
3. **Газель 3м:** цена `2000` → `1500`
4. **Газель 4.2м:** цена `2500` → `1800`
5. **Газель 5м:** была добавлена, потом **удалена** (страниц нет)
6. **`/krasnodar/index.html`:** был редирект на главную → **скопирован полноценный лендинг** из `index.html` (Краснодар) с правильными относительными путями

### Версия 1.1 — 2026-07-13 — Исправление ошибок Яндекс Вебмастера

### Ошибки, возвращенные Яндексом:

1. **Не указан обязательный атрибут в param** — `type="URL"` в `<param>` не поддерживается
2. **Не указаны обязательные параметры params** — отсутствовал `Конверсия`
3. **Неизвестный элемент** — атрибут `type` в `param` недопустим

### Исправления v1.1:
- Убран `type="URL"` из всех `param name="Ссылка на телефон"`
- Добавлен обязательный `param name="Конверсия">100` во все 40 офферов
- Исправлены названия параметров на точное соответствие Яндексу (регистр важен)

### Требования Яндекса для категории "Исполнители" — выполнены:

#### Обязательные параметры (помечены *):
| Параметр | Тип | Значение в фиде |
|----------|-----|-----------------|
| `param name="Рейтинг"` * | Число | 4.9 |
| `param name="Число отзывов"` * | Число | 150 |
| `param name="Годы опыта"` * | Число | 12 |
| `param name="Регион"` * | Строка | Краснодарский край |
| `param name="Конверсия"` * | Число | 100 |

| Параметр | Тип | Значение |
