# Журнал изменений feed.xml

## Версия 1.4 — 2026-07-14 — Исправление повторяющихся ошибок Яндекса

### Ошибки из отчёта Яндекса

1. **«Некорректно указаны исполнитель и название»** — в старой версии `<name>` одновременно содержал услугу и компанию. Исполнитель и услуга должны находиться в разных тегах.
2. **`Hatter_ReParseParamValueInvalidType`** — параметр `Ссылка на телефон` содержал `tel:+79283333281`, которое проверка Яндекса не распознала как URL.

Отчёт сформирован по старому снимку фида: в примерах присутствуют прежние URL `workers.html` и `moving.html`, а также уже отсутствующее значение `tel:+79283333281`.

### Исправления

- во всех офферах `<name>` содержит только `ООО «Правильные грузчики»`;
- каждое `<description>` начинается с явного названия услуги и города;
- необязательный параметр телефона полностью удалён, поэтому ошибка типизации `tel:` больше невозможна;
- сохранены конечные канонические URL вместо старых redirect-страниц;
- восстановлены обязательные по актуальной схеме `<sets>` и `<set-ids>`, ошибочно удалённые в промежуточной правке;
- добавлены отдельные сеты для каждой страницы услуги, чтобы имя исполнителя не повторялось внутри одного сета;
- каждому офферу назначен отдельный URL изображения в `assets/feed/` — ссылки `<picture>` больше не повторяются;
- дата каталога заменена на фактическое время формирования;
- проверены XML, обязательные поля и параметры, связи set/set-ids, уникальность ID, URL и изображений.

> В Яндекс Вебмастере следует передавать основной фид `https://pragruz.ru/feed.xml` и дождаться его повторной загрузки. Старые ошибки могут отображаться до следующего обхода.

### Версия 1.1 — 2026-07-13 — Исправление ошибок Яндекс Вебмастера

### Ошибки, возвращенные Яндексом:

1. **Не указан обязательный атрибут в param** — `type="URL"` в `<param>` не поддерживается
2. **Не указаны обязательные параметры params** — отсутствовал `Конверсия`
3. **Неизвестный элемент** — атрибут `type` в `param` недопустим

### Исправления v1.1:
- Убран `type="URL"` из всех `param name="Ссылка на телефон"`
- Добавлен обязательный `param name="Конверсия">100` во все 40 офферов
- Исправлены названия параметров на точное соответствие Яндексу (регистр важен)
